1. 表格标签

1.1 基本语法

  • <table></table> 标签：定义表格的开始与结束
  • <tr></tr> 标签：定义表格的行，嵌套在 <table></table> 中
  • <td></td> 标签：定义表格的单元格，嵌套在 <tr></tr> 中
  • <tr></tr> 中只能嵌套 <td></td>，<td></td> 中可以容纳所有标签

1.2 表格的属性

属性名 属性值 描述
border 像素值 设置表格的边框，默认 border=0 为无边框
cellspacing 像素值 设置单元格与单元格之间的空白间距，默认为2像素
cellpadding 像素值 设置单元格内容与边框之间的空白间距，默认为1像素
width 像素值 设置表格的宽度
height 像素值 设置表格的高度
align left、center、right 设置表格在网页中的水平对齐方式

1.3 合并单元格

  • 跨行合并：rowspan="合并单元格的个数"
  • 跨列合并：colspan="合并单元格的个数"
跨行合并（占2行） 单元格1 单元格2
单元格3 单元格4
跨列合并（占3列）
### 二、列表标签（List）

2. 列表标签

列表类型 标签名 说明
无序列表 <ul></ul> 里面只能嵌套 <li>，<li> 中可以嵌套任何标签，无顺序时使用较多
有序列表 <ol></ol> 里面只能嵌套 <li>，<li> 中可以嵌套任何标签，有顺序时使用
自定义列表 <dl></dl> 里面只能嵌套 <dt> 和 <dd>，<dt> 和 <dd> 中可以嵌套任何内容

  • 香蕉

  • 橙子

### 三、表单标签（Form）

3. 表单标签

使用表单是为了收集用户信息

3.1 表单域

使用 <form></form> 标签定义

属性 属性值 作用
action url地址 指定接收并处理表单数据的服务器程序地址
method get或post 设置表单数据的提交方式
name 名称 指定表单的名称，区分同一个页面中的多个表单

3.2 表单控件

3.2.1 input 控件

属性 属性值 描述
type text 单行文本输入框
password 密码输入框（内容隐藏）
radio 单选按钮（需设置相同 name 实现互斥）
checkbox 复选框（可多选）
button 普通按钮（需配合 JS 实现功能）
submit 提交按钮（将表单数据发送到服务器）
reset 重置按钮（清空表单内容）
file 文件域（用于上传文件）
hidden 隐藏域（页面不可见，用于传递隐藏数据）
name 由用户自定义 定义 input 控件的名称
value 自定义 设置控件中的默认文本值
checked checked 定义选择控件默认被选择的项（用于单选/复选框）
maxlength 正整数 定义控件允许输入的最大字符数
